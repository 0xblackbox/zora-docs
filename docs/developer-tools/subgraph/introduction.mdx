The ZORA Subgraph, is an open implementation of [The Graph Protocol](https://thegraph.com/explorer/subgraph/ourzora/zora-v1). 
It syncs with the blockchain and indexes pertinent on-chain data about ZORA's smart contracts. It also provides an easy to use GraphQL interface for developers to easily query synced data from the Ethereum blockchain.

The source code for the ZORA Subgraph is available [here](https://github.com/ourzora/zora-v1-subgraph).
