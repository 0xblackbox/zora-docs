The ZORA Subgraph, is an open implementation of [The Graph](https://thegraph.com/explorer/subgraph/ourzora/zora-v1) for the ZORA Protocol. 
It syncs with the blockchain and indexes pertinent on-chain data about ZORA's Contracts. It also provides an easy to use GraphQL interface for developers to easily query synced data from the Zora Protocol.

The source code for the ZORA Subgraph is available [here](https://github.com/ourzora/zora-v1-subgraph).
