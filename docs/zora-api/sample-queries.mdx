      saleType

### Collection Owner Count
Fetches the number of address that own an NFT in a collection.

<Tabs
  defaultValue="query"
  values={[
    { label: 'Query', value: 'query', },
    { label: 'Response', value: 'response', }
  ]
}>
<TabItem value="query">

```graphql
query NumberOfOwners {
  aggregateStat(
		query: {collectionAddresses: ["0x335eeef8e93a7a757d9e7912044d9cd264e2b2d8"]}, 
		network: {network: ETHEREUM, chain: MAINNET}, 
		statType: OWNER_COUNT
	) {
    stat {
      ... on IntValue {
        value
      }
    }
  }
}
```

### Get Token’s Ask and Transfer Events
Fetches a token with its active ZORA V3 Ask (Buy Now) and NFT Transfer events by collection address and token id.

<Tabs
  defaultValue="query"
  values={[
    { label: 'Query', value: 'query', },
    { label: 'Response', value: 'response', }
  ]
}>
<TabItem value="query">

```graphql
query TokenV3AsksAndTransfers {
  tokenMarkets(network: {network: ETHEREUM, chain: MAINNET}, pagination: {limit: 10, offset: 0}, sort: {sortKey: MINTED, sortDirection: ASC}, filter: {marketFilters: [{marketType: V3_ASK, statuses: [ACTIVE]}]}, query: {tokens: {address: "0x5180db8f5c931aae63c74266b211f580155ecac8", tokenId: "608"}}) {
    nodes {
      token {
        collectionAddress
        tokenId
      }
      markets {
        collectionAddress
        marketType
        status
        properties {
          ... on V3Ask {
            address
            askCurrency
            askPrice {
              ethPrice {
                decimal
              }
            }
            findersFeeBps
            seller
          }
        }
      }
      events(pagination: {limit: 10, offset: 0}, sort: {sortKey: CREATED, sortDirection: ASC}, filter: {eventTypes: [TRANSFER_EVENT]}) {
        nodes {
          eventType
          collectionAddress
          properties {
            ... on TransferEvent {
              fromAddress
              toAddress
            }
          }
        }
      }
    }
  }
}
```

</TabItem>
