# API Access

There are a few supported APIs to access ZORA chain information.

1. RPC Access:
    Use viem/ethers with a public RPC url or [your own node](https://github.com/conduitxyz/node) to access information.

2. Explorer Access:
    Use blockscout APIs ([rest](https://docs.blockscout.com/for-users/api/rest-api-endpoints) [graphql](https://docs.blockscout.com/for-users/api/graphql) [etherscan compat](https://docs.blockscout.com/for-users/api/rpc-endpoints)) on https://explorer.zora.energy/

3. ZORA NFT API:
    Use [api.zora.co](https://api.zora.co/) with ZORA as the network parameter to query NFT information on ZORA network.
