# Network Details 

---
### Zora Network Mainnet
Zora Network is an L2 built on top of Ethereum using the [OP stack](https://stack.optimism.io/).

| Name           | Value                             |
|----------------|-----------------------------------|
| Network        | Zora                              |
| Chain ID       | 7777777                           |
| Gas Token      | ETH                               | 
| RPC            | https://rpc.zora.co               | 
| Block Explorer | https://explorer.zora.co          |

### Zora Network Goerli Testnet

Zora Network Goerli is a Testnet L2 built on top of the Goerli Testnet.
The crypto on this network has no value and is meant for testing purposes.

| Name           | Value                             |
|----------------|-----------------------------------|
|  Network       | Zora Goerli                       |
| Chain ID       | 999                               |
| Gas Token      | ETH                               | 
| RPC            | https://testnet.rpc.zora.co       |
