# Introduction

##### Tools for collective imagination

---

Zora is a decentralized protocol where anyone can permissionlessly buy, sell, and create.
We've created a series of tools that makes it easy to get started building.

#### Our Stack

- [Smart Contracts:](../docs/smart-contracts/intro) ZORA's deployed smart contracts 

- [Nouns Builder:](../docs/smart-contracts/nouns-builder/intro) Deploy a custom DAO in minutes

- [Create:](../docs/smart-contracts/creator-tools/intro) Easy to deploy contracts for an NFT project

- [API:](../docs/zora-api/intro) Retrieve NFT metadata and market data on any ERC-721

- [ZDK:](../docs/zora-api/zdk) A development kit for interacting with ZORA's API

- [NFT Hooks:](../docs/developer-tools/nft-hooks/intro) The simplest way to interact with ZORA's API

- [NFT Rendering:](../docs/developer-tools/nft-rendering/introduction) Tools for easily displaying NFT data
