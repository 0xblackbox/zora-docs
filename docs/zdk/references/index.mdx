The Zora Development Kit (ZDK) is a tool for developers to simply and reliably interact with the Zora Protocol. The ZDK is written in Typescript and can be used in any environment in which Javascript can be run.

### Code

The source code is available on [Github](https://github.com/ourzora/zdk)

### Reference

The ZDK has four primary exports:

- [Zora](zdk/reference/zora)
- [Utils](zdk/reference/utils)
- [Addresses](zdk/reference/addresses)
- [Metadata](zdk/reference/metadata)
